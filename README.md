# Vibe Flashcard

A fun, modern flashcard web app for language learning, built with Next.js, Tailwind CSS, and daisyUI.

> **Note:** This project was created as a fun experiment to try out Cursor AI for building the entire website!

## Features
- Add your own flashcards (word + translation)
- Practice with multiple-choice questions
- Preset Mandarin cards for quick start
- Flashcards are saved in your browser (localStorage)
- Beautiful UI with daisyUI themes (currently using `cupcake`)
- Responsive and mobile-friendly

## Getting Started

### 1. Install dependencies
```bash
npm install
```

### 2. Run the development server
```bash
npm run dev
```
Visit [http://localhost:3000](http://localhost:3000) to view the app.

## Deploying to Vercel
1. Push your code to GitHub (or GitLab/Bitbucket).
2. Go to [vercel.com](https://vercel.com/), sign in, and import your repository.
3. Click **Deploy**. Vercel will handle the rest!

Alternatively, you can use the Vercel CLI:
```bash
npm i -g vercel
vercel
```
